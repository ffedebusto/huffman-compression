# huffman-compression
C++ implementation of Huffman coding from scratch. Encodes and decodes strings and binary files using a variable-length prefix tree. Covers bit manipulation, priority queues, and lossless compression fundamentals.

## What it does
- Builds a Huffman tree from input frequency analysis
- Encodes input into compressed binary
- Decodes compressed binary back to original
- Works on strings and arbitrary binary files
